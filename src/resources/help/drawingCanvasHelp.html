Dans cette partie, qui se trouve à gauche dans l'édition de projet, vous pouvez dessiner des formes. Pour ce faire, sélectionnez un élément dans la barre d'outils, et cliquez à l'endroit désiré sur le Canvas. Attention, certaines formes, comme les droites, les flèches ou les pointillés peuvent nécessiter deux clics, pour leur départ et leur fin. Les formes sont dessinés lors des clics, et le code TikZ correspondant est directement entré dans la console (voir la rubrique Console) située à droite. Pour les formes tel que les rectangles et les cercles, il est possible de les dessiner en glisser-déposer plutôt qu'en pointer-cliquer. Il est aussi possible de copier-coller une partie du dessin du projet une fois le mode "Selection" activé, en maintenant le clique gauche enfoncé et en délimitant le contenu à copier par un rectangle. Une fois la section de dessin délimité, la copie se fait avec clique gauche : Copy (ou Ctrl + c) et le collage avec clique gauche : Paste (ou Ctrl + v).
