Ce menu vous permet de retrouver un projet existant, pour modifier ses méta-informations ou pour en éditer le contenu. Plusieurs options vous sont proposées dans ce menu. Successivement, vous pouvez: - Ouvrir un projet seléctionné dans la liste de projets - Copier un projet existant, en le dupliquant - Supprimer un projet - Sauver un projet dans un autre répertoire, cette opération revient à copier ce projet vers une autre destination - Renommer un projet - Retourner au menu principal Si la liste est vide, que seul "Aucun contenu dans la table" n'est affiché, commencez par créer un fichier depuis le menu principal.
