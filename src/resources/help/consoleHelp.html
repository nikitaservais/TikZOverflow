La partie console est la partie de droite du menu d'édition. Dans le cas où vous ne la trouvez pas, vérifiez que la partie redimensionnable n'est pas complètement disposée à droite de la fenêtre. Cette partie permet d'entrer du code TikZ, ou de modifier le code généré lors de la traduction automatique générée par un dessin sur le Canvas (voir la rubrique canvas). Vous pouvez rajouter des commentaires sur le code, simplement en ne l'incluant pas dans une balise spécifique. C'est le code inscrit dans cette partie qui est stocké pour l'enregistrement d'un projet.
