L'opération de "merge" vous permet de fusionner deux branches entre elles. Lorsque les modifications apportées à une branche sont terminées et que cette version du projet vous satisfait, vous pouvez fusionner la branche actuelle avec une autre de votre choix. Généralement, le "merge" se fait vers la branche "master" car celle-ci contient ce que vous désirez dans la version finale de votre projet, mais libre à vous de faire ce que vous voulez ! Une fois la fusion effectuée, vous retrouverez toutes les informations dans la branche vers laquelle vous avec "merge".
