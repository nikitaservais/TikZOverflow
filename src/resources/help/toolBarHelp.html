La barre d'outils se trouve dans la partie supérieure de l'édition de projet. Elle permet de séléctionner des pinceaux, leur taille et leur couleur, pour dessiner sur le Canvas (voir la rubrique Canvas). En plus de ces boutons de pinceaux, deux boutons se trouvent à l'extrême droite: - Le bouton de traduction, qui permet de traduire le code de la console en diagrammes dans le Canvas - Le bouton d'effacage, qui vide le Canvas de ses dessins. Le code TikZ n'est quant à lui pas détruit. - Le bouton preview, qui permet d'obtenir un aperçu du projet sous format pdf. - Le bouton Selection Mode, qui permet de passer en mode copier-coller.
